# 常见问题

## 连接设置

### 输入主机地址时需要包含 http/s 前缀吗？

不需要。因为 Comfy Portal 同时使用 HTTP 协议和 WebSocket 与服务器通信。系统会自动处理协议前缀 - 您只需要输入域名或 IP 地址即可。

### 我无法连接到我的服务器，应该如何排查？

1. 首先请确认你可以在设备上通过浏览器访问服务器
2. 如果可以但仍不能通过 app 连接，尝试调整 "Use SSL" 的选项并刷新

### 我无法通过 comfy-portal-endpoint 插件同步我的 workflow 到 app？

1. 请确保你有一个活跃的浏览器窗口链接到服务器，这是由于 workflow flow 的转换依赖于前端代码，而我们目前没有更好的解决方案
2. 请确保 workflow 是可以正常在该服务器上运行的，否则可能无法正确识别部分 node 导致同步失败
