# 常见问题

## 连接设置

### 输入主机地址时需要包含 http/s 前缀吗？

不需要。因为 Comfy Portal 同时使用 HTTP 协议和 WebSocket 与服务器通信。系统会自动处理协议前缀 - 您只需要输入域名或 IP 地址即可。

## 性能

### 为什么生成图片后下载这么慢？

下载速度主要取决于您的网络连接质量。如果您使用的是本地 WiFi 网络，可以尝试以下优化建议：

- 确保运行 ComfyUI 服务器的设备靠近 WiFi 路由器
- 保持您的手机靠近 WiFi 路由器
- 尽量减少路由器和设备之间的物理障碍物
